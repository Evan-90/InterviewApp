这是一个基于 [Create React App](https://github.com/facebookincubator/create-react-app)的项目.


## 本地安装

- 首先你需要克隆项目到你到本地环境

- 进入项目根目录你需要这些操作：
  - [npm start]
  - [npm run eject]
- 你可以看见这样的文件目录：
my-app/
  README.md
  node_modules/
  package.json
  public/
    index.html
    favicon.ico
  src/
    componets/
    container/
    img/
    redux/
    util/
    config.js
    index.js
    reducer.js
```

- 项目打包：
  - [npm run build]
